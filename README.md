# 📋 FocusFlow – Smart Productivity Dashboard

A modern productivity dashboard built using HTML, CSS, and JavaScript to help users organize tasks, track productivity, and stay focused.

## 🌐 Live Demo

https://anitakulagod.github.io/FocusFlow/

---

## ✨ Features

- ✅ Add, Edit & Delete Tasks
- 🔍 Search Tasks
- 📂 Filter Tasks (All / Pending / Completed)
- 📊 Dashboard Statistics
- 📈 Progress Tracking
- 📅 Due Date Management
- 🚩 Priority Levels
- 🍅 Pomodoro Focus Timer
- 🌙 Dark Mode
- 💾 Local Storage
- 📊 Productivity Analytics (Chart.js)
- 📤 Export Tasks to CSV
- 🔔 Toast Notifications
- 🗑 Delete Confirmation Modal
- 💡 Daily Motivation Quotes
- 📱 Responsive Design

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
- Chart.js
